Introdução

Dando início a esta documentação, apresentamos o projeto com o intuito de esclarecer sua finalidade e objetivos. Este empreendimento busca aprimorar a experiência de pesquisa por informações dos usuários aprofundada e ágil. Neste contexto, propomos um aumento significativo na velocidade de investigação, permitindo um acesso mais eficiente e preciso às informações necessárias aos utilizadores. A seguir, apresentaremos detalhadamente as etapas, metodologias e recursos envolvidos, buscando fornecer uma compreensão clara e abrangente sobre o desenvolvimento desta iniciativa.

Requisitos

  • Ter XAMPP instalado.
  • Ter Visual Studio Code instalado.
  • Ter o devido banco de dados.

Instalação

Passos necessários para instalar e configurar o projeto:

  1. Instalar XAMPP. XAMPP
  2. Instalar Visual Studio Code. Visual Studio Code
  3. Instalar o devido banco de dados. Banco de Dados
  4. Entrar nesse Repositório. Repositório
  5. Com tudo instalado, abra os arquivos e use.

Uso

Principais recursos disponíveis:

  • Login e Senha
  • Cadastro
  • Tela com todas as informações dos usuarios
  • Botão para editar
  • Botão para excluir
  • Botão para voltar
  • Barra de Pesquisa
  • Armazenamento no banco de dados

Configuração

Descreva as opções de configuração disponíveis e como elas podem ser personalizadas:

  • Opção 1: Edição das informações dos usuários.
  • Opção 2: Remoção das informações dos usuários.
  • Opção 3: Pesquisa das informações dos usuários.

Contribuição

  1. Aumento de velocidade para realizar uma pesquisa das informações dos usuarios.
  2. Organização com as informações dos usuarios.
